# Studio+ — Entrega vendável (MVP)

## O que esta versão entrega
- Setup inicial para criar o primeiro admin pela interface (`/setup`)
- Login por role (`admin` e `client_owner`)
- Painel admin
- Cadastro completo de cliente com criação de usuário no Auth
- Painel do cliente
- Página pública por slug
- Solicitação pública de agendamento
- Aprovação/recusa de solicitação
- Agenda com atualização de status
- Configurações básicas do negócio
- Logout

## Como vender esta versão
Venda como:
- mini site + agendamento + painel da clínica/salão
- implantação inicial + mensalidade de manutenção
- solução para pequenos studios, lash, sobrancelha, manicure, estética

## O que ainda é evolução futura (não impede vender como MVP)
- recuperação de senha
- cobrança recorrente automática
- upload real de logo/capa/galeria
- múltiplas profissionais com agenda individual
- relatórios avançados
- WhatsApp oficial/API
- edição completa em modal de clientes/serviços
- analytics e auditoria

## Posicionamento honesto
Esta entrega está pronta para:
- demonstração
- implantação em clientes-piloto
- venda como MVP premium

Para vender como SaaS maduro em escala, a próxima fase é produto/QA/financeiro/observabilidade.
